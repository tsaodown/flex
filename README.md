# Usage
<!-- usage -->
```sh-session
$ npm install -g flex
$ flex COMMAND
running command...
$ flex (-v|--version|version)
flex/0.0.0 darwin-x64 node-v7.10.0
$ flex --help [COMMAND]
USAGE
  $ flex COMMAND
...
```

## `flex help [COMMAND]`

display help for flex

```
USAGE
  $ flex help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
